# 01: 数组：长度属于类型

数组是固定长度、元素类型相同的连续数据集合。在 Go 里，数组长度是类型的一部分。

---

## 1. 声明数组

```go
var a [3]int
fmt.Println(a) // [0 0 0]
```

`[3]int` 表示长度为 3 的 int 数组。数组元素会被初始化为元素类型的零值。

常见零值：

- `int`: `0`
- `bool`: `false`
- `string`: `""`
- 指针、slice、map、函数、channel、interface: `nil`

---

## 2. 初始化数组

```go
nums := [3]int{1, 2, 3}
names := [2]string{"alice", "bob"}
```

可以用 `...` 让编译器根据元素个数推断长度：

```go
nums := [...]int{1, 2, 3}
fmt.Printf("%T\n", nums) // [3]int
```

也可以指定下标初始化：

```go
nums := [5]int{
    0: 10,
    3: 40,
}

fmt.Println(nums) // [10 0 0 40 0]
```

---

## 3. 长度属于类型

`[3]int` 和 `[4]int` 是两个不同类型。

```go
var a [3]int
var b [4]int

// a = b // 编译错误：类型不同
```

这一点和很多语言不同。Go 的数组不是“可变长度容器”，它更像一个固定大小的值。

---

## 4. 访问与越界

```go
nums := [3]int{10, 20, 30}

fmt.Println(nums[0]) // 10
fmt.Println(nums[2]) // 30
```

下标从 `0` 开始。访问不存在的位置会出错：

```go
// fmt.Println(nums[3]) // 编译期或运行期报错
```

如果下标是常量，编译器通常能直接发现越界。如果下标来自变量，越界会在运行时触发 panic。

---

## 5. 遍历数组

普通 `for`：

```go
nums := [3]int{10, 20, 30}

for i := 0; i < len(nums); i++ {
    fmt.Println(i, nums[i])
}
```

`range`：

```go
for i, v := range nums {
    fmt.Println(i, v)
}
```

`i` 是下标，`v` 是元素值的副本。后面讲 `range` 时会单独展开这个复制语义。

---

## 6. 数组赋值会复制整个数组

数组是值。赋值时会复制整个数组。

```go
a := [3]int{1, 2, 3}
b := a

b[0] = 100

fmt.Println(a) // [1 2 3]
fmt.Println(b) // [100 2 3]
```

修改 `b` 不会影响 `a`。

---

## 7. 数组作为函数参数

函数参数也是值传递。把数组传入函数时，会复制整个数组。

```go
func change(nums [3]int) {
    nums[0] = 100
}

func main() {
    a := [3]int{1, 2, 3}
    change(a)
    fmt.Println(a) // [1 2 3]
}
```

如果要修改原数组，可以传数组指针：

```go
func change(nums *[3]int) {
    nums[0] = 100
}

func main() {
    a := [3]int{1, 2, 3}
    change(&a)
    fmt.Println(a) // [100 2 3]
}
```

日常业务代码里，更常见的是用 slice 表示一段可变长度的数据。

---

## 8. 多维数组

多维数组可以理解为“数组的元素仍然是数组”。

```go
matrix := [2][3]int{
    {1, 2, 3},
    {4, 5, 6},
}

fmt.Println(matrix[0][1]) // 2
```

遍历：

```go
for row := 0; row < len(matrix); row++ {
    for col := 0; col < len(matrix[row]); col++ {
        fmt.Print(matrix[row][col], " ")
    }
    fmt.Println()
}
```

---

## 9. 什么时候使用数组

数组适合长度固定且长度本身有意义的场景：

- 固定大小的哈希值，如 `[32]byte`。
- 固定维度的坐标，如 `[3]float64`。
- 底层实现中需要确定大小的数据块。

大多数普通列表、集合、查询结果、请求数据，应该优先使用 slice。
