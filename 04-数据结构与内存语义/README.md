本模块讲 Go 最常用的复合数据结构：数组、slice、map、struct 和指针。学习重点不只是会写语法，还要能判断一段代码到底复制了什么、共享了什么、修改了哪里。

## 学习目标

完成本模块后，你应该能够：

- 说清楚数组和 slice 的区别，理解数组长度为什么属于类型。
- 理解 slice 头、底层数组、`len`、`cap`、`append` 之间的关系。
- 判断两个 slice 是否共享底层数组，以及修改一个 slice 会不会影响另一个 slice。
- 正确创建、读取、更新、删除和遍历 map。
- 理解 map 变量赋值、传参、元素更新和遍历顺序的语义。
- 使用 struct 表达数据模型，理解字段、零值、字面量、标签和比较规则。
- 判断方法应该使用值接收者还是指针接收者。
- 正确使用指针表达可变状态、避免无意义复制，并知道 nil 指针的风险。
- 推演 `range` 在数组、slice、map、string 上的复制行为。

---

## 学习建议

这一章建议边读边画内存关系：

1. 变量名指向哪个值。
2. 赋值或传参时复制了哪一层。
3. 多个变量是否共享同一个底层数组或 map 数据。
4. 修改发生在副本上，还是发生在共享的数据上。
5. `append` 之后原来的 slice 是否还应该继续使用。
