# 09: 函数与错误综合练习

这一组练习用于串联函数定义、多返回值、错误、defer、panic/recover。

---

## 练习1: 阶乘函数

实现函数：

```go
func factorial(n int) (int, error)
```

要求：

1. `n < 0` 时返回错误。
2. `0` 和 `1` 返回 `1`。
3. 正常返回阶乘结果。
4. 调用方必须检查错误。

---

## 练习2: 安全除法

实现函数：

```go
func safeDivide(a, b float64) (float64, error)
```

要求：

1. `b == 0` 时返回错误。
2. 正常返回计算结果和 `nil`。
3. 调用方打印错误或结果。

---

## 练习3: 文件读取

实现函数：

```go
func readFile(path string) ([]byte, error)
```

要求：

1. 使用 `os.Open` 打开文件。
2. 使用 `defer` 关闭文件。
3. 使用 `io.ReadAll` 读取内容。
4. 所有错误都要带上下文。

---

## 练习4: 错误判断

实现函数：

```go
func readRequiredFile(path string) ([]byte, error)
```

要求：

1. 文件不存在时返回包装后的错误。
2. 调用方使用 `errors.Is(err, os.ErrNotExist)` 判断文件不存在。
3. 打印不同的错误提示。

---

## 练习5: 计数器闭包

实现函数：

```go
func makeCounter(start int) func() int
```

要求：

1. 第一次调用返回 `start + 1`。
2. 每次调用都在上一次基础上加 `1`。
3. 创建两个 counter，确认它们互不影响。

---

## 验证建议

每个练习都建议运行：

```bash
go run .
```

对于纯函数，可以尝试写测试：

```bash
go test ./...
```
