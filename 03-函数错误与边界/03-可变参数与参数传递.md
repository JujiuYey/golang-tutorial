# 03: 可变参数与参数传递

Go 函数参数默认是值传递。理解这一点，有助于判断函数会不会修改调用方的数据。

---

## 1. 可变参数

可变参数使用 `...`：

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}
```

调用：

```go
fmt.Println(sum(1, 2, 3))
fmt.Println(sum(1, 2, 3, 4, 5))
```

在函数内部，`nums` 的类型是 `[]int`。

---

## 2. 传入 slice

已有 slice 可以用 `...` 展开：

```go
nums := []int{1, 2, 3}
fmt.Println(sum(nums...))
```

---

## 3. 值传递

普通参数传入函数时，会复制一份。

```go
func change(n int) {
    n = 100
}

func main() {
    x := 1
    change(x)
    fmt.Println(x) // 1
}
```

函数内部修改 `n`，不会影响外部 `x`。

---

## 4. slice 参数

slice 值本身会被复制，但它指向同一个底层数组。

```go
func change(nums []int) {
    nums[0] = 100
}

func main() {
    values := []int{1, 2, 3}
    change(values)
    fmt.Println(values) // [100 2 3]
}
```

slice 参数本身仍然是值传递。传入函数时复制的是 slice 头，复制后的 slice 头仍指向同一个底层数组，所以修改元素会影响调用方看到的数据。

---

## 5. map 参数

map 作为参数传递时，函数内部修改 map 内容会影响外部。

```go
func change(m map[string]int) {
    m["count"] = 100
}

func main() {
    data := map[string]int{"count": 1}
    change(data)
    fmt.Println(data)
}
```

---

## 6. 指针参数

如果要让函数修改一个普通变量，可以传指针。

```go
func change(n *int) {
    *n = 100
}

func main() {
    x := 1
    change(&x)
    fmt.Println(x) // 100
}
```

指针会在后面的数据结构章节继续讲。

---

## 练习

实现两个函数：

```go
func addOne(n int)
func addOnePtr(n *int)
```

要求：

1. 调用 `addOne(x)` 后，外部 `x` 不变。
2. 调用 `addOnePtr(&x)` 后，外部 `x` 加 `1`。
3. 解释这两个函数行为不同的原因。
