# 04: 函数类型、匿名函数与闭包

Go 中函数也是值。函数可以赋给变量，可以作为参数，也可以作为返回值。

---

## 1. 函数类型

```go
func add(a, b int) int {
    return a + b
}

func main() {
    var op func(int, int) int
    op = add

    fmt.Println(op(3, 4))
}
```

`func(int, int) int` 就是函数类型。

---

## 2. 匿名函数

匿名函数没有名字，可以直接赋给变量。

```go
func main() {
    add := func(a, b int) int {
        return a + b
    }

    fmt.Println(add(3, 4))
}
```

也可以立即调用：

```go
func main() {
    result := func(a, b int) int {
        return a + b
    }(3, 4)

    fmt.Println(result)
}
```

---

## 3. 函数作为参数

```go
func apply(op func(int, int) int, a, b int) int {
    return op(a, b)
}

func main() {
    add := func(a, b int) int {
        return a + b
    }

    fmt.Println(apply(add, 3, 4))
}
```

这种写法常用于回调、策略、遍历处理等场景。

---

## 4. 函数作为返回值

```go
func getOperator(op string) func(int, int) int {
    switch op {
    case "+":
        return func(a, b int) int { return a + b }
    case "-":
        return func(a, b int) int { return a - b }
    default:
        return func(a, b int) int { return 0 }
    }
}
```

调用：

```go
add := getOperator("+")
fmt.Println(add(3, 4))
```

---

## 5. 闭包

闭包是可以访问外部变量的函数。

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

func main() {
    c := counter()
    fmt.Println(c()) // 1
    fmt.Println(c()) // 2
    fmt.Println(c()) // 3
}
```

`count` 在 `counter` 返回后仍然被返回的函数持有。

---

## 6. 闭包常见用途

闭包适合保存少量状态：

```go
func makePrefixer(prefix string) func(string) string {
    return func(s string) string {
        return prefix + s
    }
}
```

调用：

```go
addID := makePrefixer("ID-")
fmt.Println(addID("100"))
```

## 练习

实现函数：

```go
func makeCounter(start int) func() int
```

要求：

1. 第一次调用返回 `start + 1`。
2. 每次调用都在上一次基础上加 `1`。
3. 创建两个 counter，确认它们互不影响。
