# 01: 函数定义、参数与返回值

函数是 Go 组织逻辑的基本单位。一个函数的签名描述了它需要什么输入、返回什么输出。

---

## 1. 基本函数定义

```go
func greet() {
    fmt.Println("Hello")
}
```

函数由 `func` 关键字、函数名、参数列表、返回值和函数体组成。

```go
func add(a int, b int) int {
    return a + b
}
```

相邻参数类型相同时，可以合并：

```go
func add(a, b int) int {
    return a + b
}
```

---

## 2. 无返回值函数

```go
func printUser(name string, age int) {
    fmt.Printf("%s is %d years old\n", name, age)
}
```

无返回值函数通常用于执行动作，比如打印、写文件、发送请求等。

---

## 3. 单返回值函数

```go
func square(n int) int {
    return n * n
}
```

调用：

```go
func main() {
    result := square(5)
    fmt.Println(result)
}
```

---

## 4. 函数签名

函数签名包括参数类型和返回值类型。

```go
func add(a, b int) int
```

下面两个函数签名不同：

```go
func addInt(a, b int) int
func addFloat(a, b float64) float64
```

Go 不支持函数重载。不能在同一个包里定义两个同名函数，即使参数类型不同。

```go
// func add(a, b int) int { return a + b }
// func add(a, b float64) float64 { return a + b } // 编译错误
```

---

## 5. return

`return` 用于结束函数并返回结果。

```go
func abs(n int) int {
    if n < 0 {
        return -n
    }
    return n
}
```

函数遇到 `return` 后立即结束。

---

## 6. 参数名只在函数内有效

```go
func double(n int) int {
    return n * 2
}
```

参数 `n` 的作用域只在 `double` 函数内部。

## 练习

实现两个函数：

```go
func max(a, b int) int
func isEven(n int) bool
```

要求：

1. `max` 返回两个整数中较大的一个。
2. `isEven` 判断数字是否为偶数。
3. 在 `main` 中调用并打印结果。
