## 1. 函数基础

### 1.1 基本函数定义

```go
// 无参数无返回值
func greet() {
    fmt.Println("Hello!")
}

// 有参数
func greet(name string) {
    fmt.Println("Hello,", name)
}

// 有参数有返回值
func add(a int, b int) int {
    return a + b
}

// 简化参数类型
func add(a, b int) int {
    return a + b
}
```

### 1.2 多返回值

Go函数支持多返回值，这是Go的特色之一：

```go
// 返回两个值
func divide(a, b int) (int, int) {
    return a / b, a % b
}

// 调用
quotient, remainder := divide(10, 3)  // 3, 1
```

### 1.3 命名返回值

```go
func calc(a, b int) (sum int, product int) {
    sum = a + b      // 直接赋值给命名返回值
    product = a * b
    return           // 不用写返回值
}

// 调用
s, p := calc(3, 4)
```

## 2. 函数高级特性

### 2.1 可变参数

```go
// 可变参数函数
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

sum(1, 2, 3)        // 6
sum(1, 2, 3, 4, 5)  // 15
```

### 2.2 函数作为参数

```go
// 函数作为参数
func apply(op func(int, int) int, a, b int) int {
    return op(a, b)
}

add := func(a, b int) int {
    return a + b
}

result := apply(add, 3, 4)  // 7
```

### 2.3 函数作为返回值

```go
// 函数作为返回值
func getOperator(op string) func(int, int) int {
    switch op {
    case "+":
        return func(a, b int) int { return a + b }
    case "-":
        return func(a, b int) int { return a - b }
    default:
        return func(a, b int) int { return 0 }
    }
}

add := getOperator("+")
fmt.Println(add(3, 4))  // 7
```

### 2.4 闭包

```go
// 闭包：函数内部定义的函数，可以访问外部变量
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
fmt.Println(c())  // 3
```

## 3. 错误处理

### 3.1 error接口

Go使用error接口表示错误：

```go
type error interface {
    Error() string
}
```

### 3.2 创建error

```go
import "errors"

// 方式1: errors.New
err := errors.New("文件不存在")

// 方式2: fmt.Errorf
import "fmt"
err := fmt.Errorf("无效的参数: %s", arg)
```

### 3.3 错误处理模式

```go
// 标准错误处理
result, err := doSomething()
if err != nil {
    // 处理错误
    return nil, err
    // 或者 log.Fatal(err)
}
// 使用result
```

### 3.4 错误示例

```go
import (
    "fmt"
    "os"
    "errors"
)

func readFile(name string) ([]byte, error) {
    data, err := os.ReadFile(name)
    if err != nil {
        return nil, fmt.Errorf("读取文件失败: %w", err)
    }
    return data, nil
}
```

### 3.5 忽略错误

```go
// 使用 _ 忽略错误（谨慎使用）
result, _ := divide(10, 2)  // 忽略错误
```

## 4. defer与panic/recover

### 4.1 defer延迟执行

`defer` 语句会延迟函数调用，常用于资源清理：

```go
func read() {
    file, err := os.Open("test.txt")
    if err != nil {
        return
    }
    
    defer file.Close()  // 函数返回前执行
    
    // ... 使用file读写
}  // file.Close() 在这里自动执行
```

### 4.2 defer执行时机

```go
func example() {
    fmt.Println("开始")
    
    defer fmt.Println("defer 1")
    defer fmt.Println("defer 2")
    defer fmt.Println("defer 3")
    
    fmt.Println("结束")
    // 输出: 开始 -> 结束 -> defer 3 -> defer 2 -> defer 1
}
```

### 4.3 defer与参数求值

```go
func main() {
    i := 1
    defer fmt.Println(i)  // 参数立即求值，打印1
    i = 2
    return
}
```

### 4.4 panic恐慌

`panic` 会中断程序执行：

```go
func panicExample() {
    fmt.Println("开始")
    panic("发生严重错误")
    fmt.Println("不会执行")  // 不会到这里
}
```

### 4.5 recover恢复

`recover` 可以捕获panic，防止程序崩溃：

```go
func safeCall() {
    defer func() {
        if r := recover(); r != nil {
            fmt.Println("捕获到panic:", r)
        }
    }()
    
    panic("故意的panic")
    fmt.Println("不会执行")
}
```

### 4.6 典型用法

```go
func main() {
    defer fmt.Println("程序结束")
    
    fmt.Println("开始执行")
    
    // 模拟错误处理
    defer func() {
        if r := recover(); r != nil {
            fmt.Println("恢复:", r)
        }
    }()
    
    panic("测试panic")
}
```

## 5. 练习作业

### 练习1: 阶乘函数

```go
package main

import "fmt"

func factorial(n int) (int, error) {
    if n < 0 {
        return 0, fmt.Errorf("负数没有阶乘")
    }
    if n == 0 || n == 1 {
        return 1, nil
    }
    result := 1
    for i := 2; i <= n; i++ {
        result *= i
    }
    return result, nil
}

func main() {
    for i := 0; i <= 10; i++ {
        f, err := factorial(i)
        if err != nil {
            fmt.Println("错误:", err)
        } else {
            fmt.Printf("%d! = %d\n", i, f)
        }
    }
}
```

### 练习2: 文件读取

```go
package main

import (
    "fmt"
    "os"
)

func readFile(name string) ([]byte, error) {
    file, err := os.Open(name)
    if err != nil {
        return nil, fmt.Errorf("打开文件失败: %w", err)
    }
    
    defer file.Close()  // 确保关闭
    
    stat, err := file.Stat()
    if err != nil {
        return nil, fmt.Errorf("获取文件信息失败: %w", err)
    }
    
    data := make([]byte, stat.Size())
    _, err = file.Read(data)
    if err != nil {
        return nil, fmt.Errorf("读取文件失败: %w", err)
    }
    
    return data, nil
}

func main() {
    data, err := readFile("test.txt")
    if err != nil {
        fmt.Println("错误:", err)
        return
    }
    fmt.Println(string(data))
}
```

### 挑战题: 安全除法

```go
package main

import (
    "errors"
    "fmt"
)

func safeDivide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("除数不能为零")
    }
    return a / b, nil
}

func main() {
    tests := []struct {
        a, b float64
    }{
        {10, 2},
        {10, 0},
        {9, 2},
    }
    
    for _, test := range tests {
        result, err := safeDivide(test.a, test.b)
        if err != nil {
            fmt.Printf("%.2f / %.2f = Error: %v\n", test.a, test.b, err)
        } else {
            fmt.Printf("%.2f / %.2f = %.2f\n", test.a, test.b, result)
        }
    }
}
```
