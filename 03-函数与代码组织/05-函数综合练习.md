# 05: 函数综合练习

这 5 道练习只使用本章已经学过的函数知识。目标不是处理失败，而是把函数签名、多返回值、参数传递、函数值和闭包真正写顺手。

---

## 本篇重点（模块回顾）

- 函数签名写清参数类型和返回值类型。
- Go 支持多返回值，不需要先塞进数组或对象。
- 参数传入函数时会复制；究竟复制了什么，要结合具体类型判断。
- 函数可以放进变量、作为参数传入，也可以作为返回值交出去。
- 闭包能继续使用外层函数中的变量，并保留自己的状态。

---

## 练习 1：矩形计算

实现函数：

```go
func rectangle(width, height int) (area int, perimeter int)
```

要求返回面积和周长，并用 `3, 4` 验证结果。

---

## 练习 2：交换两个值

实现函数：

```go
func swap(a, b string) (string, string)
```

调用 `swap("left", "right")`，确认两个返回值的位置已经交换。

---

## 练习 3：可变参数求和

实现函数：

```go
func sum(nums ...int) int
```

分别调用 `sum()`、`sum(1, 2, 3)` 和 `sum(2, 4, 6, 8)`，确认零个、三个和四个参数都能正常工作。

---

## 练习 4：把函数作为参数

实现函数：

```go
func apply(a, b int, operation func(int, int) int) int
```

分别传入加法函数和乘法函数。重点观察：`apply` 不关心具体怎么算，它只负责调用收到的函数。

---

## 练习 5：计数器闭包

实现函数：

```go
func makeCounter(start int) func() int
```

要求：

1. 第一次调用返回 `start + 1`。
2. 每次调用都在上一次基础上加 `1`。
3. 创建两个 counter，确认它们互不影响。

提示：被保留下来的计数变量应该声明在外层函数里，但要放在返回的匿名函数外面。
