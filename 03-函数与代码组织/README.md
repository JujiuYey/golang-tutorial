# 03: 函数与代码组织

前两章写的代码大多从上到下挤在 `main` 里。这一章只解决一件事：**怎样把一段逻辑收进函数，再通过参数、返回值和函数值把它们组合起来**。

这里暂时不讲 `error`。Go 的 `error` 本身是接口，而接口要等学完 struct、方法和接口之后才真正说得清。错误处理会放到第 06 章单独学习。

## 学习目标

完成本模块后，你应该能够：

- 熟练定义函数，理解参数、返回值和函数签名。
- 正确使用多返回值、命名返回值和可变参数。
- 理解 Go 的参数是值传递，以及不同类型被复制时发生了什么。
- 使用函数类型、匿名函数和闭包表达简单抽象。
