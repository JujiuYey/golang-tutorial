Go Module 使用

初始化项目

mkdir myproject

cd myproject

go mod init module名

go.mod 文件结构

module github.com/user/myproject

go 1.22

require (

github.com/gin-gonic/gin v1.9.1

github.com/go-sql-driver/mysql

)

常用命令

go mod tidy # 整理依赖

go get xxx # 添加依赖

go mod download # 下载依赖

⚠️ 注意事项

• go.mod 使用UTF-8编码

• 模块名通常为仓库路径

• proxy.golang.org 官方代理
