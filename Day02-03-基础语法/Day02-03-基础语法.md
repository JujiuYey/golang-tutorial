## 1. 变量与常量

### 1.1 变量声明

Go语言有多种变量声明方式：

```go
// 方式1: var 声明
var name string = "Go"
var age int = 18

// 方式2: 类型推断（省略类型）
var name = "Go"        // 自动推断为 string
var age = 18           // 自动推断为 int

// 方式3: 简短声明（函数内部使用）
name := "Go"
age := 18

// 方式4: 多变量声明
var a, b, c int = 1, 2, 3
x, y := 10, 20
```

### 1.2 变量作用域

```go
package main

import "fmt"

// 全局变量
var globalVar = "全局"

func main() {
    // 局部变量
    localVar := "局部"
    fmt.Println(globalVar)  // OK
    fmt.Println(localVar)   // OK
}
```

### 1.3 常量声明

```go
// 单个常量
const PI = 3.14159
const APP_NAME = "MyApp"

// 批量声明
const (
    STATUS_OK    = 200
    STATUS_ERROR = 500
    STATUS_NOT_FOUND = 404
)

// iota 枚举
const (
    MONDAY = iota  // 0
    TUESDAY        // 1
    WEDNESDAY      // 2
    THURSDAY       // 3
    FRIDAY         // 4
    SATURDAY       // 5
    SUNDAY         // 6
)

// iota 自增值
const (
    A = iota  // 0
    B         // 1
    C         // 2
)
```

### 1.4 注意事项

| 规则 | 说明 |
|------|------|
| `:=` 只能在函数内使用 | 全局变量必须用 `var` |
| 变量必须使用 | 声明后不使用会报错 |
| 常量不能修改 | `const` 声明后值不可变 |
| 命名规则 | 驼峰命名，首字母大写可导出 |

## 2. 数据类型

### 2.1 整型

```go
// 有符号整型
var i int   = 10       // 平台相关（32/64位）
var i8 int8 = 127      // -128 ~ 127
var i16 int16 = 32767  // -32768 ~ 32767
var i32 int32 = 2147483647
var i64 int64 = 9223372036854775807

// 无符号整型
var u uint  = 10       // 平台相关
var u8 uint8 = 255    // 0 ~ 255
var u16 uint16 = 65535
var u32 uint32
var u64 uint64

// 别名
var b byte = 255       // byte = uint8
var r rune = 12345     // rune = int32 (表示Unicode)
```

### 2.2 浮点型

```go
var f1 float32 = 3.14
var f2 float64 = 3.141592653589793  // 默认类型

// 自动推断
f := 3.14  // float64
```

### 2.3 复数

```go
var c1 complex64 = 3 + 4i
var c2 complex128 = 5 + 12i

// 函数
fmt.Println(real(c2))  // 5.0
fmt.Println(imag(c2))   // 12.0
```

### 2.4 布尔型

```go
var isActive bool = true
isValid := false

// 布尔值不能转换为整型
// var n int = bool(1)  // 错误！
```

### 2.5 字符串

```go
var str string = "Hello, Go!"
s := "Hello World"

// 字符串操作
len(str)                    // 长度
str + " Welcome"            // 拼接
str[0]                      // 第一个字符 (byte)
str[0:5]                    // 切片

// 多行字符串
multiline := `
第一行
第二行
第三行
`
```

### 2.6 类型转换

**Go必须显式转换，不能隐式自动转换！**

```go
var i int = 100
var f float64 = float64(i)  // int -> float64
var s string = strconv.Itoa(i)  // int -> string

// 字符串转数字
num, err := strconv.Atoi("123")

// string转[]byte
b := []byte("hello")

// []byte转string
s := string([]byte{104, 101})
```

## 3. 运算符

### 3.1 算术运算符

```go
a, b := 10, 3

fmt.Println(a + b)  // 13 加
fmt.Println(a - b)  // 7  减
fmt.Println(a * b)  // 30 乘
fmt.Println(a / b)  // 3  整除
fmt.Println(a % b)  // 1  取余

a++  // 自增
a--  // 自减
```

### 3.2 关系运算符

```go
a, b := 10, 20

fmt.Println(a == b)  // false 相等
fmt.Println(a != b)  // true  不等
fmt.Println(a > b)   // false 大于
fmt.Println(a < b)   // true  小于
fmt.Println(a >= b)  // false
fmt.Println(a <= b)  // true
```

### 3.3 逻辑运算符

```go
x, y := true, false

fmt.Println(x && y)  // false 逻辑与
fmt.Println(x || y)  // true  逻辑或
fmt.Println(!x)       // false 逻辑非
```

### 3.4 位运算符

```go
a, b := 5, 3  // 5=101, 3=011

fmt.Println(a & b)   // 1   按位与: 001
fmt.Println(a | b)   // 7   按位或: 111
fmt.Println(a ^ b)   // 6   异或:  110
fmt.Println(^a)      // -6  取反
fmt.Println(a << 2)  // 20  左移2位
fmt.Println(a >> 1)  // 2   右移1位
```

### 3.5 赋值运算符

```go
a := 10

a += 5   // a = a + 5  -> 15
a -= 3   // a = a - 3  -> 12
a *= 2   // a = a * 2  -> 24
a /= 4   // a = a / 4  -> 6
a %= 4   // a = a % 4  -> 2
```

## 4. 流程控制

### 4.1 if 条件判断

```go
// 基本if
if score >= 60 {
    fmt.Println("及格")
}

// if-else
if score >= 90 {
    fmt.Println("优秀")
} else {
    fmt.Println("一般")
}

// if-else if-else
if score >= 90 {
    fmt.Println("A")
} else if score >= 80 {
    fmt.Println("B")
} else if score >= 60 {
    fmt.Println("C")
} else {
    fmt.Println("D")
}

// 初始化语句（作用域仅在if内）
if a := 10; a > 5 {
    fmt.Println(a)  // 10
}
// fmt.Println(a)  // 错误，a未定义
```

### 4.2 for 循环

Go没有while，用for实现：

```go
// 基本for循环
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// 省略初始值
i := 0
for i < 5 {
    i++
}

// 省略所有（死循环）
for {
    // 无限循环，需要配合 break/return
    break
}

// for-range 遍历
nums := []int{1, 2, 3, 4, 5}
for index, value := range nums {
    fmt.Printf("索引: %d, 值: %d\n", index, value)
}

// 省略索引或值
for _, value := range nums {
    fmt.Println(value)
}
for i := range nums {
    fmt.Println(i)
}
```

### 4.3 switch 分支

```go
// 基本switch
day := 3
switch day {
case 1:
    fmt.Println("周一")
case 2:
    fmt.Println("周二")
case 3:
    fmt.Println("周三")
default:
    fmt.Println("其他")
}

// 多值匹配
switch day {
case 1, 2, 3, 4, 5:
    fmt.Println("工作日")
case 6, 7:
    fmt.Println("周末")
}

// 初始化 + 条件
switch score := 85; {
case score >= 90:
    fmt.Println("A")
case score >= 80:
    fmt.Println("B")
default:
    fmt.Println("C")
}

// fallthrough: 继续执行下一个case
switch 2 {
case 1:
    fmt.Println("one")
    fallthrough
case 2:
    fmt.Println("two")
    fallthrough
case 3:
    fmt.Println("three")
}
// 输出: two, three
```

### 4.4 跳转语句

```go
// break: 跳出当前循环
for i := 0; i < 5; i++ {
    if i == 3 {
        break
    }
    fmt.Println(i)  // 0, 1, 2
}

// continue: 跳过本次循环
for i := 0; i < 5; i++ {
    if i == 2 {
        continue
    }
    fmt.Println(i)  // 0, 1, 3, 4
}

// goto: 跳转到标签
goto label
fmt.Println("不会执行")
label:
fmt.Println("跳转到这里")
```

### 4.5 嵌套循环 + break/continue

```go
// break只跳出最近一层循环
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if j == 1 {
            break  // 只跳出内层循环
        }
        fmt.Println(i, j)
    }
}

// 跳出指定层（使用标签）
outer:
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if j == 1 {
            break outer  // 跳出外层循环
        }
        fmt.Println(i, j)
    }
}
```

## 5. 练习作业

### 练习1: 猜数字游戏

```go
package main

import (
    "fmt"
    "math/rand"
    "time"
)

func main() {
    rand.Seed(time.Now().UnixNano())
    target := rand.Intn(100) + 1  // 1-100随机数
    
    fmt.Println("猜数字游戏！")
    fmt.Println("我已经想好了一个1-100的数字")
    
    for {
        var guess int
        fmt.Print("请输入你的猜测: ")
        fmt.Scan(&guess)
        
        if guess > target {
            fmt.Println("太大了！")
        } else if guess < target {
            fmt.Println("太小了！")
        } else {
            fmt.Println("恭喜你，猜对了！")
            break
        }
    }
}
```

### 练习2: 九九乘法表

```go
package main

import "fmt"

func main() {
    for i := 1; i <= 9; i++ {
        for j := 1; j <= i; j++ {
            fmt.Printf("%dx%d=%d\t", j, i, i*j)
        }
        fmt.Println()
    }
}
```

### 练习3: 简易计算器

```go
package main

import "fmt"

func main() {
    var a, b float64
    var op string
    
    fmt.Print("请输入第一个数: ")
    fmt.Scan(&a)
    fmt.Print("请输入运算符 (+ - * /): ")
    fmt.Scan(&op)
    fmt.Print("请输入第二个数: ")
    fmt.Scan(&b)
    
    var result float64
    switch op {
    case "+":
        result = a + b
    case "-":
        result = a - b
    case "*":
        result = a * b
    case "/":
        if b != 0 {
            result = a / b
        } else {
            fmt.Println("除数不能为0")
            return
        }
    default:
        fmt.Println("无效的运算符")
        return
    }
    
    fmt.Printf("%.2f %s %.2f = %.2f\n", a, op, b, result)
}
```
