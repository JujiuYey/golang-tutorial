数据类型

整型

int, int8, int16, int32, int64

uint, uint8, uint16...

byte = uint8

rune = int32

浮点型

float32 32位

float64 64位(默认)

complex64, complex128

布尔型

bool

true / false

不能转换为int

字符串

string: UTF-8编码的字符序列

name := "Hello Go"

类型转换

Go必须显式转换，不能自动转换

int64(a) / float64(i)

var i int = 100

var f float64 = float64(i) // 显式转换

str := strconv.Itoa(i) // int转string
