# 08: 表单、文件上传与静态文件

除了 JSON，HTTP 服务还经常处理表单、文件上传和静态文件。

---

## 1. 解析普通表单

```go
func submit(w http.ResponseWriter, r *http.Request) {
    if err := r.ParseForm(); err != nil {
        http.Error(w, "invalid form", http.StatusBadRequest)
        return
    }
    name := r.Form.Get("name")
    fmt.Fprintln(w, name)
}
```

`ParseForm` 会解析 URL query 和表单 body。

---

## 2. PostForm

```go
if err := r.ParseForm(); err != nil {
    return
}
name := r.PostForm.Get("name")
```

`PostForm` 只包含 body 中的表单数据，不包含 URL query。

---

## 3. 限制请求体大小

```go
r.Body = http.MaxBytesReader(w, r.Body, 10<<20)
```

上传接口一定要限制大小。否则恶意或异常请求可能占用大量内存和磁盘。

---

## 4. 文件上传

```go
func upload(w http.ResponseWriter, r *http.Request) {
    r.Body = http.MaxBytesReader(w, r.Body, 10<<20)

    if err := r.ParseMultipartForm(10 << 20); err != nil {
        http.Error(w, "invalid multipart form", http.StatusBadRequest)
        return
    }

    file, header, err := r.FormFile("file")
    if err != nil {
        http.Error(w, "missing file", http.StatusBadRequest)
        return
    }
    defer file.Close()

    fmt.Fprintf(w, "uploaded: %s\n", header.Filename)
}
```

不要盲目信任客户端提供的文件名。

---

## 5. 保存上传文件

```go
dst, err := os.CreateTemp("uploads", "upload-*")
if err != nil {
    http.Error(w, "create file failed", http.StatusInternalServerError)
    return
}
defer dst.Close()

if _, err := io.Copy(dst, file); err != nil {
    http.Error(w, "save file failed", http.StatusInternalServerError)
    return
}
```

使用服务端生成的文件名，减少路径穿越风险。

---

## 6. 静态文件服务

```go
fs := http.FileServer(http.Dir("public"))
mux.Handle("/static/", http.StripPrefix("/static/", fs))
```

请求 `/static/app.css` 会映射到 `public/app.css`。

---

## 7. 下载文件

```go
func download(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Disposition", `attachment; filename="report.txt"`)
    http.ServeFile(w, r, "reports/report.txt")
}
```

如果文件路径来自用户输入，要严格限制在允许目录内，避免读取任意文件。

---

## 8. Multipart 客户端

构造上传请求常用 `mime/multipart`。

```go
var body bytes.Buffer
writer := multipart.NewWriter(&body)

part, err := writer.CreateFormFile("file", "hello.txt")
if err != nil {
    return err
}
if _, err := part.Write([]byte("hello")); err != nil {
    return err
}
if err := writer.Close(); err != nil {
    return err
}

req, err := http.NewRequest(http.MethodPost, rawURL, &body)
if err != nil {
    return err
}
req.Header.Set("Content-Type", writer.FormDataContentType())
```

---

## 9. 文件名安全

```go
name := filepath.Base(header.Filename)
```

`filepath.Base` 可以去掉路径部分，但仍然要考虑重名、特殊字符、扩展名限制、内容类型校验等问题。

更稳的做法是服务端生成存储名，把原始文件名只作为元数据保存。

---

## 10. 表单处理检查

1. body 是否限制大小。
2. 上传文件是否关闭。
3. 文件名是否可信。
4. 保存路径是否可控。
5. 错误状态码是否清楚。

---

## 练习

1. 写 handler 读取普通表单字段。
2. 写文件上传 handler，限制 10MB。
3. 保存上传文件时使用 `os.CreateTemp`。
4. 配置 `/static/` 静态文件服务。
5. 用 `mime/multipart` 构造上传请求测试。
