# Learning Log

这个目录记录用户在本 Go 教程仓库里的学习画像和学习日志。它是 agent 的长期上下文，不是课程正文。

## 文件结构

- `profile.md`：长期学习画像，记录当前水平、偏好、薄弱点和下一步。
- `daily/YYYY-MM-DD.md`：每日学习日志。只有用户确认后，agent 才能追加。

## 使用规则

1. 每次学习辅导开始前，agent 先读 `profile.md`。
2. 如果用户提到最近学习进度，agent 再读最近的 daily 日志。
3. 一次只推进 1 个认知点，不连续追问多个问题。
4. 学习小节结束时，agent 先生成日志草稿。
5. 用户确认后，agent 才能写入 daily 日志，并按需更新 `profile.md`。

## Daily 日志模板

```md
## HH:MM - 主题

- 学习内容：
- 用户当前理解：
- 卡点或易错点：
- 已验证命令：
- 下一步：
```
