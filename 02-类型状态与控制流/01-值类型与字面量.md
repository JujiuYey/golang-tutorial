# 01: 值、类型与字面量

Go 是静态类型语言。每个值都有类型，编译器会在程序运行前检查类型是否匹配。

本篇要解决三个问题：

1. Go 里常见的值类型有哪些？
2. 字面量是什么？
3. 编译器如何根据字面量推断默认类型？

---

## 1. 值与类型

值是程序处理的数据，类型决定这个值能参与哪些操作。

```go
package main

import "fmt"

func main() {
    age := 18
    name := "Go"
    active := true

    fmt.Printf("%T %v\n", age, age)
    fmt.Printf("%T %v\n", name, name)
    fmt.Printf("%T %v\n", active, active)
}
```

输出：

```text
int 18
string Go
bool true
```

`%T` 打印类型，`%v` 打印值。初学时可以经常用它们确认自己的判断。

---

## 2. 基本类型概览

| 类型 | 说明 | 示例 |
|------|------|------|
| `bool` | 布尔值，只能是 `true` 或 `false` | `ok := true` |
| `string` | 字符串 | `name := "golang"` |
| `int` | 常用整数，位数依赖平台 | `count := 10` |
| `int8` / `int16` / `int32` / `int64` | 指定位数有符号整数 | `var id int64 = 1001` |
| `uint` / `uint8` / `uint16` / `uint32` / `uint64` | 无符号整数 | `var n uint = 10` |
| `byte` | `uint8` 的别名，常表示字节 | `var b byte = 'A'` |
| `rune` | `int32` 的别名，常表示 Unicode 码点 | `var r rune = '你'` |
| `float32` / `float64` | 浮点数 | `price := 19.9` |
| `complex64` / `complex128` | 复数 | `z := 1 + 2i` |

日常 Go 代码里最常见的是 `bool`、`string`、`int`、`int64`、`float64`、`byte`、`rune`。

---

## 3. 字面量

字面量是直接写在代码里的值。

```go
func main() {
    fmt.Println(123)       // 整数字面量
    fmt.Println(3.14)      // 浮点数字面量
    fmt.Println("hello")   // 字符串字面量
    fmt.Println('你')      // rune 字面量
    fmt.Println(true)      // 布尔字面量
}
```

字面量本身可以是无类型的，只有在赋给变量、参与表达式、传给函数时，编译器才会确定它的具体类型。

```go
func main() {
    i := 123    // 默认推断为 int
    f := 3.14   // 默认推断为 float64
    r := '你'   // 默认推断为 rune，也就是 int32

    fmt.Printf("%T\n", i)
    fmt.Printf("%T\n", f)
    fmt.Printf("%T\n", r)
}
```

---

## 4. 类型推断

`:=` 会让编译器根据右边的值推断类型：

```go
func main() {
    n := 10       // int
    f := 3.14     // float64
    s := "hello"  // string

    fmt.Printf("%T\n", n)
    fmt.Printf("%T\n", f)
    fmt.Printf("%T\n", s)
}
```

类型推断不是动态类型。推断完成后，变量类型就固定了。

```go
func main() {
    n := 10
    // n = "hello" // 编译错误：不能把 string 赋给 int
}
```

---

## 5. 类型决定可用操作

不同类型支持的操作不同。

```go
func main() {
    fmt.Println(1 + 2)
    fmt.Println("Go" + "lang")
    // fmt.Println("Go" - "lang") // 编译错误
}
```

编译器拒绝 `"Go" - "lang"`，因为字符串没有减法操作。

## 练习

先判断每个变量的类型，再运行验证。

```go
package main

import "fmt"

func main() {
    a := 1
    b := 1.0
    c := "1"
    d := '1'
    e := true

    fmt.Printf("a: %T %v\n", a, a)
    fmt.Printf("b: %T %v\n", b, b)
    fmt.Printf("c: %T %v\n", c, c)
    fmt.Printf("d: %T %v\n", d, d)
    fmt.Printf("e: %T %v\n", e, e)
}
```

问题：

1. `a` 和 `b` 的类型一样吗？
2. `c` 和 `d` 的类型一样吗？
3. 为什么 `'1'` 打印出来的值不是字符串 `"1"`？
