# 02: 变量声明、赋值、零值与作用域

变量是状态的名字。读 Go 代码时，要能看出变量在哪里声明、在哪里可见、什么时候被修改。

---

## 1. var 声明

`var` 是最基础的变量声明方式。

```go
var name string
var age int
var ok bool
```

可以声明时赋值：

```go
var name string = "Go"
var age int = 18
```

也可以让编译器推断类型：

```go
var name = "Go"
var age = 18
```

批量声明常用于包级变量或一组相关变量：

```go
var (
    appName = "demo"
    version = "1.0.0"
    debug   = true
)
```

---

## 2. 短变量声明

函数内部可以使用 `:=`：

```go
func main() {
    name := "Go"
    age := 18

    fmt.Println(name, age)
}
```

`:=` 只能在函数内部使用，不能用于包级变量。

```go
// name := "Go" // 编译错误：函数外不能使用 :=
```

---

## 3. 赋值与多变量赋值

声明后修改变量，用 `=`。

```go
func main() {
    count := 1
    count = 2
    fmt.Println(count)
}
```

Go 支持多变量赋值：

```go
func main() {
    a, b := 1, 2
    a, b = b, a
    fmt.Println(a, b) // 2 1
}
```

这在交换变量值时很常见。

---

## 4. 短变量声明的重复规则

`:=` 左边至少要有一个新变量。

```go
func main() {
    name := "Go"
    age := 18

    name, city := "Gopher", "Beijing" // city 是新变量，所以合法
    fmt.Println(name, age, city)
}
```

如果左边没有任何新变量，就会编译失败：

```go
func main() {
    name := "Go"
    // name := "Gopher" // 编译错误：没有新变量
    name = "Gopher"
    fmt.Println(name)
}
```

---

## 5. 零值

Go 变量声明后一定有值。没有显式赋值时，会得到对应类型的零值。

| 类型 | 零值 |
|------|------|
| `int` / `int64` / `float64` | `0` |
| `bool` | `false` |
| `string` | `""` |
| 指针、slice、map、channel、function | `nil` |

```go
package main

import "fmt"

func main() {
    var count int
    var name string
    var ok bool

    fmt.Printf("count=%d name=%q ok=%v\n", count, name, ok)
}
```

零值是 Go 设计里的重要概念。很多类型会尽量设计成零值可用。

---

## 6. 作用域

作用域决定变量在哪里可见。

```go
package main

import "fmt"

var global = "global"

func main() {
    local := "local"

    if ok := true; ok {
        msg := "inside if"
        fmt.Println(global, local, msg)
    }

    // fmt.Println(ok)  // 编译错误：ok 只在 if 语句内有效
    // fmt.Println(msg) // 编译错误：msg 只在 if 代码块内有效
}
```

---

## 7. 变量遮蔽

内层作用域可以声明和外层同名的变量，这叫变量遮蔽。

```go
package main

import "fmt"

func main() {
    count := 10

    if true {
        count := 20
        fmt.Println("inside:", count)
    }

    fmt.Println("outside:", count)
}
```

输出：

```text
inside: 20
outside: 10
```

需要修改外层变量时，用 `=`：

```go
func main() {
    count := 10

    if true {
        count = 20
    }

    fmt.Println(count) // 20
}
```

---

## 8. 未使用变量

Go 不允许声明变量后不使用。

```go
func main() {
    // name := "Go" // 编译错误：declared and not used
}
```

临时忽略某个值时，可以用空白标识符 `_`：

```go
func main() {
    name := "Go"
    _ = name
}
```

`_` 应该谨慎使用。它能让代码通过编译，但也可能掩盖你本该处理的值。

## 练习

不要先运行，先预测输出。

```go
package main

import "fmt"

func main() {
    count := 1

    if true {
        count := count + 1
        fmt.Println("inside:", count)
    }

    fmt.Println("outside:", count)
}
```

问题：

1. 这段代码能编译吗？
2. 如果能，输出是什么？
3. `if` 里面的 `count := count + 1` 创建了几个 `count`？
