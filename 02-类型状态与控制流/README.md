## 学习目标

完成本模块后，你应该能够：

- 说清楚 Go 常见值类型、字面量和默认类型。
- 熟练使用 `var`、`:=`、多变量赋值和批量声明。
- 理解零值、作用域、变量遮蔽和未使用变量的编译错误。
- 掌握常量、无类型常量和 `iota` 的基本用法。
- 处理整数、浮点数、布尔值、字符串、byte、rune 的常见问题。
- 理解 Go 为什么要求显式类型转换。
- 推演 `if`、`for`、`range`、`switch`、`break`、`continue`、标签的执行路径。
- 能完成基础语法综合练习，并解释每个选择背后的语言规则。

---

## 学习建议

每篇文章都建议按这个顺序学习：

1. 先读规则。
2. 预测示例代码能不能编译、输出什么。
3. 自己运行代码验证。
4. 修改代码，观察编译器错误或输出变化。
5. 完成练习，并用自己的话解释原因。
