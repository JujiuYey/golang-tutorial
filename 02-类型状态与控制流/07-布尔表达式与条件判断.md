# 07: 布尔表达式与条件判断

条件判断的核心是布尔表达式。Go 的 `if` 条件必须是 `bool`，不能用数字或字符串代替。

---

## 1. 关系运算符

```go
func main() {
    a, b := 10, 20

    fmt.Println(a == b)
    fmt.Println(a != b)
    fmt.Println(a > b)
    fmt.Println(a < b)
    fmt.Println(a >= b)
    fmt.Println(a <= b)
}
```

关系运算的结果是 `bool`。

---

## 2. 逻辑运算符

```go
func main() {
    x, y := true, false

    fmt.Println(x && y) // false
    fmt.Println(x || y) // true
    fmt.Println(!x)     // false
}
```

`&&` 表示同时满足，`||` 表示满足任意一个，`!` 表示取反。

---

## 3. if 条件必须是 bool

Go 不允许下面这种写法：

```go
func main() {
    n := 1
    // if n { // 编译错误
    //     fmt.Println("ok")
    // }
}
```

必须写成明确的布尔表达式：

```go
if n != 0 {
    fmt.Println("ok")
}
```

---

## 4. if / else if / else

```go
func grade(score int) string {
    if score >= 90 {
        return "A"
    } else if score >= 80 {
        return "B"
    } else if score >= 60 {
        return "C"
    }
    return "D"
}
```

Go 里 `else` 必须和前一个 `}` 在同一行：

```go
if score >= 60 {
    fmt.Println("pass")
} else {
    fmt.Println("fail")
}
```

---

## 5. if 初始化语句

`if` 可以带初始化语句，变量只在这个 `if` 结构内有效。

```go
func main() {
    if n := len("golang"); n > 3 {
        fmt.Println(n)
    }

    // fmt.Println(n) // 编译错误
}
```

---

## 6. 拆分复杂条件

复杂条件建议拆成有名字的变量：

```go
func main() {
    score := 86
    attendance := 0.9

    passedScore := score >= 60
    passedAttendance := attendance >= 0.8

    if passedScore && passedAttendance {
        fmt.Println("pass")
    }
}
```

这样更容易检查条件是否写反。

## 练习

实现一个函数：

```go
func canRegister(age int, agreedTerms bool) bool
```

要求：

1. 年龄必须大于等于 `18`。
2. 必须同意条款。
3. 两个条件都满足才返回 `true`。
