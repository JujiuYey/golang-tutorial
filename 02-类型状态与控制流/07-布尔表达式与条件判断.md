# 07: 布尔表达式与条件判断

条件判断的核心是布尔表达式。Go 的 `if` 条件必须是 `bool`，不能用数字或字符串代替。

---

## 1. 关系运算符

```go
func main() {
    a, b := 10, 20

    fmt.Println(a == b)
    fmt.Println(a != b)
    fmt.Println(a > b)
    fmt.Println(a < b)
    fmt.Println(a >= b)
    fmt.Println(a <= b)
}
```

关系运算的结果是 `bool`。

---

## 2. 逻辑运算符

```go
func main() {
    x, y := true, false

    fmt.Println(x && y) // false
    fmt.Println(x || y) // true
    fmt.Println(!x)     // false
}
```

`&&` 表示同时满足，`||` 表示满足任意一个，`!` 表示取反。

---

## 3. if 条件必须是 bool

Go 不允许下面这种写法：

```go
func main() {
    n := 1
    // if n { // 编译错误
    //     fmt.Println("ok")
    // }
}
```

必须写成明确的布尔表达式：

```go
if n != 0 {
    fmt.Println("ok")
}
```

---

## 4. if / else if / else

```go
func grade(score int) string {
    if score >= 90 {
        return "A"
    } else if score >= 80 {
        return "B"
    } else if score >= 60 {
        return "C"
    }
    return "D"
}
```

Go 里 `else` 必须和前一个 `}` 在同一行：

```go
if score >= 60 {
    fmt.Println("pass")
} else {
    fmt.Println("fail")
}
```

---

## 5. if 初始化语句

`if` 可以带初始化语句，变量只在这个 `if` 结构内有效。

```go
func main() {
    if n := len("golang"); n > 3 {
        fmt.Println(n)
    }

    // fmt.Println(n) // 编译错误
}
```

---

## 6. if 初始化语句的最常见应用：err-handling 模式

第 5 节用 `len("golang")` 演示了 if 初始化语句的语法。在真实 Go 代码里，**这个特性几乎全部用在 error handling 上**。

### 语法拆解

```go
if err := doSomething(); err != nil {
    return err
}
```

三部分：

- `err := doSomething()` —— 初始化（声明 + 赋值）
- `err != nil` —— 条件
- `;` —— 分隔初始化和条件

### 核心好处：作用域限制

`err` 只在 `if/else` 块里活，出 `{}` 就消失。**这跟函数参数一样——用完即弃，不污染外层 scope。**

所以 Go 代码里 `if err != nil` 满屏飞是正常的。Go 函数通常返回 `(value, err)`，调用方立刻用 `if err != nil` 处理掉。

### 示例

```go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    if n, err := strconv.Atoi("42"); err == nil {
        fmt.Println("parsed:", n)
    } else {
        fmt.Println("error:", err)
    }
}
```

输出：`parsed: 42`。

---

## 7. 拆分复杂条件

复杂条件建议拆成有名字的变量：

```go
func main() {
    score := 86
    attendance := 0.9

    passedScore := score >= 60
    passedAttendance := attendance >= 0.8

    if passedScore && passedAttendance {
        fmt.Println("pass")
    }
}
```

这样更容易检查条件是否写反。
